## Markdown Syntax
| Syntax | Element |
| --- | --- |
| `#` | heading 1 |
| `##-n` | heading 2-n |
| `-` | unordered list item |
| `1.-n.` | odered list item |
| `[][x]` | checkbox |
| `**...**` | bold |
| `_..._` | italic |
| `[link name](https://)` | link |
| `>` | blockquote |
| `---` | divider |
| ` `...` ` | inline code block |
| ` ```...``` ` | code block |
